# EOS Connecting 🌐

Sitio web oficial de EOS Connecting.

## 🚀 Stack
- HTML5 / CSS3 / JavaScript
- i18n ready (5 idiomas: ES, EN, DE, PT, IT)
- Desplegado en Vercel

## 📁 Estructura
- `/pages` - Páginas internas del sitio
- `/css` - Estilos globales y responsive
- `/js` - Lógica de internacionalización y navegación
- `/data` - Archivos de traducción JSON
- `/assets` - Imágenes y recursos
